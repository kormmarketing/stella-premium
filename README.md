# Стелла Память — Мемориальные комплексы

Премиальный лендинг на React + Vite + Tailwind + Framer Motion.

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Сборка

```bash
npm run build
```

## Замена изображений

Замените `public/placeholder.svg` или добавьте в `public/`:
- `hero.jpg` — главный экран (широкий кадр)
- `project1.jpg`, `project2.jpg` — проекты
- `stone-1.jpg` — `stone-4.jpg` — материалы

Обновите пути в `src/App.jsx`.

## Контакты

Измените `PHONE` и `WHATSAPP` в начале `src/App.jsx`.
